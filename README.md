# 공주대 수강신청 사이트 제작
2024.10 ~ 2024.12(약 5주)

3학년 1학기 소프트웨어공학 강의 시간에 설계한 수강신청 사이트를 기준으로

3학년 2학기 데이터베이스프로그래밍 강의의 프로젝트로 수강신청 사이트를 제작합니다.


## 주요 기능
1. **회원 가입** 이후 **세션 기반 로그인**이 가능합니다.
2. 메인 화면에서 **본인의 학적 정보를 확인**할 수 있으며 **각종 공지를 확인**하고 해당 공지 사이트로 이동할 수 있습니다.
3. 사용자가 원하는 정보를 기입하여 해당 정보에 맞게 **필터링된 개설 강좌를 조회**할 수 있습니다.
4. **장바구니 기능**을 이용하여 원하는 강좌를 미리 담아놓고 이후 수강신청 시 사용할 수 있습니다.
5. **수강신청** 시 장바구니에 담아놓았던 강좌를 바로 신청하거나, 원하는 강좌를 조회해 신청할 수 있습니다.
6. 장바구니 내역과 수강신청 내역을 각각 **조회**할 수 있고 **수정 및 취소** 할 수 있습니다.

## 개발 환경
- Language : Java(17)
- BackEnd Framework : Spring Boot 3.4.0
- Database : MySQL
- ORM : MyBatis
- FrontEnd : Thymeleaf, HTML/CSS/ JavaScript
